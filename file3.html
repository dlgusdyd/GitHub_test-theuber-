GitHub 원격저장소에 커밋을 로컬저장소에 내려받기

1.파일을 내려받을 폴더를 생성 후 우클릭를 통해 Git Bash Here를 실행

2.GitHub의 원격저장소에 들어가 Clone or download 버튼을 클릭해 주소를 복사

-->Download ZIP를 통해 소스코드를 똑같이 받을 수 있지만 그럴 경우 원격저장소와 버전 정보가 제외되니 꼭 클론을 통해 다운

3.git clone 주소값 입력 / 한간 띄고 . 꼭 삽입 / 폴더 안에 새로운 폴더가 생기게 되므로 폴더 구조가 복잡해질수 있다.

4.git push origin master 를 통해 원격저장소에 푸쉬

원격저장소의 새로운 커밋을 로컬저장소에 갱신하기

1.갱신하고자 하는 폴더 우클릭 후 Git Bash Here 실행

2.git pull origin master를 통해 새로운 커밋파일을 로컬저장소에 다운

-->여기서 pull이란 원격저장소에 새로운 커밋이 있다면 그걸 내 로컬저장소에 받아오라는 명령어이다.
