로컬저장소 생성하기

1.로컬저장소를 만들고자 하는 폴더를 우클릭 선택 후 Git Bash Here 선택

2.Git 명령어인 git init 입력 후 로컬저장소 생성 -> 해당폴더에 git이라는 폴더와 함께 Git Bash Here program에 "Initialized empty Git ~~"라는 텍스트가 나오면 성공

커밋 하는 방법

1.GitHub user.name / user.email를 등록

--> git config --global user.name "name명"

--> git config --global user.email "email명"

2.커밋할 파일명을 입력 git add index.html 입력 후 git 명령어를 통해 해당 파일의 수정내용 이나 설명을 추가할수있다.

--> git commit -m "설명추가" / 1 file changed , 1 insertion(+) 라는 내용이 나오면 성공

다른 커밋으로 시간 여행하기

1.git log를 통해 내가 커밋한 내용을 확인 할 수 있다.입력 후 나오는 commit 이후에 8자리를 확인 후 git checkout 8자리 입력 하면 해당 커밋으로 돌아갈 수 있다.

--> Head is now at 8자리 사이트 설명 추가 라는 텍스트가 나오면 성공

2.최신 커밋으로 돌아가고 싶으면 git checkout - 를 입력하면 최신 커밋으로 돌아갈 수 있다. 여기서 - 는 최신 커밋을 의미한다.
